Kỹ thuật media query sẽ cho phép css chọn những mã css nào
    áp dụng cho màn hình có kích cỡ nào
    Ví dụ :
    - màn hình từ 0px đến 600px sẽ áp dụng css cho body màu vàng
    - màn hình từ 601px đến 900px sẽ áp dụng css cho body màu xanh
    - màn hình từ 901px đến 1200px sẽ áp dụng css cho body màu tím
    - màn hình từ 1200px sẽ áp dụng css cho body màu đỏ

    @media screen and (min-width: 600px){
        // css cho màn hình từ 600px trở lên ( 600 đến vô cực )
    }

    @media screen and (max-width: 600px){
        // css cho màn hình từ 600px trở xuống ( 0 đến 600 )
    }
