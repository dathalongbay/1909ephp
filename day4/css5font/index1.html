font-family: kiểu font

    trong css, có 2 kiểu tên font family :
    kiểu 1 : tên chung - 1 nhóm các kiểu font có kiểu hiển thị tương tự nhau
    ( ví dụ serif và monospace )
    kiểu 2 : 1 tên font cụ thể ( ví dụ "time new roman" hoặc "arial" )

    ví dụ thẻ p có phần css như sau :
    font-family: "Times New Roman", Times, serif;
    * chú ý 1 : khi viết code css cho thuộc tính font-family nếu tên font mà
    nhiều hơn 1 từ thì sẽ phải đặt tên font trong ""
    ví dụ "Times New Roman" nếu tên font chỉ có 1 từ duy nhất thì ko cần dùng ""
    * chú ý 2 : khi khai báo kiểu font của 1 selector trong css
    thì sẽ khai báo nhiều kiểu font theo thứ tự
    font-family: "Times New Roman", Times, serif;
    ví dụ máy tính tìm "Times New Roman" để hiển thị
    nếu font này không có thì tìm tiếp font Times
    trong TH Times trên máy ko tồn tại
    thì máy tính sẽ tìm 1 font bất kỳ thuộc nhóm font serif để hiển thị
    và quy tắc là đặt tên font cụ thể đầu tiều , đặt tên theo kiểu nhóm như là
    serif hoặc san-serif ở cuối cùng
