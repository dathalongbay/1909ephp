Tạo ra 1 giao diện có 1 hàng chứa 5 cột
        Kết luận :
        class css .col tạo ra giao diện 1 cột trong css
        khi 1 hàng muốn có bao nhiêu cột bằng nhau thì mình sẽ sử dụng bấy
        nhiều thẻ có class là .col
        ví dụ
        muốn có giao diện 3 cột => sử dụng 3 thẻ div có class là .col trong class .row
        muốn có giao diện 4 cột => sử dụng 4 thẻ div có class là .col trong class .row
        muốn có giao diện 5 cột => sử dụng 5 thẻ div có class là .col trong class .row
        muốn có giao diện 2 cột => sử dụng 2 thẻ div có class là .col trong class .row
        lưu ý không nhất thiết phải sử dụng thẻ div mà có thể dùng thẻ khác section miễn
        là các thẻ này có class css là .col
        Chú ý : class .col này không hỗ trợ tốt responsive sẽ sử dụng class
        như .col-xs .col-sm ... mà sẽ học ở bài sau
        class .col chỉ sử dụng khi mà muốn tạo ra giao diện có nhiều cột bằng nhau
