Vòng lặp trong js

        Cú pháp của vòng lặp for :

        for (statement 1; statement 2; statement 3) {
          // code block to be executed
        }
        Statement 1 sẽ được thực thi 1 lần trước khi đoạn code trong {} được chạy
        => thường được sử dụng để khai báo biến đếm với giá trị ban đầu
        Statement 2 khai báo điều kiện vòng lặp sẽ chạy
        => statement 2 là mệnh đề điều kiện
        Statement 3 sẽ được thực thi mỗi lần sau khi khối code {} được chạy
        => sự thay đổi của biến đếm sau mỗi lần chạy

        *** Khi code vòng lặp có 1 điều phải chú ý
        Vòng lặp vô hạn được hiểu là 1 vòng lặp chạy mãi không kết thúc
        kết quả trình duyệt sẽ bị treo
