Cách 3 nhúng css vào html ( external css )

    Nhúng thông qua thẻ link của html trong thẻ head
    thẻ link có rel="stylesheet" và href="đường dẫn của file css"

    - 2 loại đường dẫn để liên kết file vào trong html
    + đường dẫn tuyệt đối : đường dẫn theo ổ đĩa, theo domain
    https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css
    + đường dẫn tương đối :
    nếu truy cập ra folder bên ngoài sử dụng ../
    mỗi 1 lần back ra ngoài 1 cấp thì sẽ sử dụng ../
    ví dụng nếu file css nằm ở folder bên ngoài 2 cấp : ../../

    Còn nếu truy cập vào cấp bên trong đi theo đường dẫn của folder
