abstract class & interface
        abstract class ( class trừu tượng )
        - trong abstract class vừa có abstract method
        và các method thông thường
        - phương thức abstract có chỉ code khai báo tên của phương thức
        mà không có code thực thi
        ** nhưng class con kế thừa class abstract phải tiến hành
        viết code thực thi cụ thể cho các abstract method trong abstract class
        - interface ( giao diện )
        khi 1 class implements ( thực thi ) 1 interface cụ thể
        thì class đó phải viết code thực thi cụ thể cho các method bên trong
        interface
        Các method trong interface thì chỉ phần khai báo tên phương thức
        mà không có code thực thi

        abstract class ( là gì )
        interface ( làm gì )
