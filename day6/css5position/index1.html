thuộc tính position ( vị trí )
        có 1 số các giá trị sau cho thuộc tính này
        giá trị 1 : static ( tĩnh ) đây là giá trị mặc định
        giá trị 2 : relative ( tương đối )
        giá trị 3 : fixed ( cố định )
        giá trị 4 : absolute ( tuyệt đối )
        giá trị 5 : sticky


        *** static ( mặc định )
        position: static không chịu ảnh hưởng của các thuộc tính top, bottom , left , right
thẻ html có position là static thì luôn hiển thị vị trí theo kiểu mặc định
