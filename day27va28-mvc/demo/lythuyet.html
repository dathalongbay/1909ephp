Khi chạy 1 file php
        trong ứng dụng thêm sửa xóa lấy dữ liệu
        ví dụ
        index.php ( hiển thị bảng danh sách trong CSDL )
        create.php ( hiển thị form tạo bản ghi mới )
        edit.php ( hiển thị form sửa bản ghi )
        delete.php ( xóa bản ghi )
        khi chạy ứng dụng code thuần thì sẽ chạy riêng lẻ từng file php khác nhau
        => Nhưng trong ứng dụng mvc thì chỉ 1 đầu vào duy nhất
        là file index.php
        và index.php sẽ gọi các file khác để xử lý tác vụ
